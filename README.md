# Gushwork.assignment.sriyaram

This project is developed as part of the Gushwork Frontend Assignment.  
The implementation strictly follows the provided Figma design and is built using only:

- HTML5
- CSS3 (Flexbox & Grid)
- Vanilla JavaScript

## Features

- Sticky header that appears after scrolling past the first fold
- Smooth scroll-based animations
- Interactive image carousel
- Image zoom effect on hover
- Fully responsive layout (Desktop, Tablet, Mobile)
- Semantic HTML structure
- Clean and organized code

## Project Structure

gushwork-assignment/
│
├── index.html
├── styles.css
├── script.js
├── assets/
└── README.md

## How to Run

1. Download or clone the repository
2. Open `index.html` in any modern browser

No external frameworks or libraries were used.
